# 🛡️ Log Analyzer — Purple Team Tool

Анализатор логов безопасности для поиска атакующих IP-адресов.

## 🚀 Возможности
- Парсинг строк логов
- Подсчёт атак по IP и портам
- Вывод самого активного злоумышленника

## 📊 Пример вывода
Всего событий: 10
Самый активный IP: 5.5.5.5 (атак: 5)

## 🛠 Как запустить
python log_analyzer.py
